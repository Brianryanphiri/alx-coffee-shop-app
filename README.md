# ☕ ALX Coffee Shop App

A mobile coffee ordering application built with **React Native (Expo)** and **NativeWindCSS**.  
This project is part of the **ALX React Native UI Implementation Challenge**.  

The app replicates a Figma design for a modern **Coffee Shop** mobile experience, focusing on **pixel-perfect UI**, smooth navigation, and a clean developer workflow.

---

## 📱 Features
- 🏠 **Home Screen** – Browse featured coffees and categories  
- 🔍 **Search Screen** – Find your favorite drinks easily  
- ☕ **Coffee Details Screen** – View product details, size options, and pricing  
- 🛒 **Cart Screen** – Manage selected items before checkout  
- 👤 **Profile / Login Screen** – User authentication UI  

---

## 🛠️ Tech Stack
- **React Native** (Expo with TypeScript)  
- **NativeWindCSS** (Tailwind CSS for React Native)  
- **React Navigation** (stack & bottom tabs)  
- **Expo Asset Management** (fonts, images, icons)  

---

## 🚀 Getting Started

### Prerequisites
- Node.js >= 18  
- Expo CLI (`npm install -g expo-cli`)  
- Git

### Installation
```bash
# Clone the repo
git clone https://github.com/Brianryanphiri/alx-coffee-shop-app

# Navigate into the project
cd alx-coffee-shop-app

# Install dependencies
npm install

# Run the app
npx expo start
